\section{句式}

元音结尾句提问：以=v结尾接=n，其余接\bil{=o}{\lat{-u}}。

是：\A (oqi) \B inu.

不是：\A (oqi) \B waka.

存在 / 有： …… bi. 如batu taqikv de bi.

不存在 / 没有： …… akv. 如batu taqikv de akv. 此外akv可用于否定形容词，如batu \bil{ahvn}{哥哥} \aux{qi} \bil{den}{高} akv.

\A 与\B 一起：\A \B -i/ni emgi/sasa.

尚未做：\V =r=A unde.

没做过：\V =HA akv 等价：\V =HA=kv

曾做过：\V =HA bihe

正在做：\V =me bi

曾在做：\V =me bihe

形容词和动词表并且：\lat{adj./v.} bime \lat{adj./v.}，两侧可以混用形容词和动词。

称为\A 的\B ： \A sere \B 

COD和COI的配合：\A \B be \C de \bil{bumbi}{给}.中，\B 为COD，\C 为COI。

\subsection{使被动}

使动：\A \B \bil{be}{让} [\C (be)] \V =bumbi. \A 让 \B 做 \V \C 。

被动：\A \B \bil{de}{被} (\C) \V =bumbi. \C 极少出现

使被动：\A \B \bil{be}{让 / 把} \C \bil{de}{被 / 给} \V =bumbi. 

\begin{itemize}
    \item ama batu \aux{be} efen jebumbi.
    \item efen bati \aux{de} jebumbi.
    \item ama efen \aux{be} batu \aux{de} jebumbi. 爸爸把点心给巴图吃 / 爸爸使点心被巴图吃。
\end{itemize}
